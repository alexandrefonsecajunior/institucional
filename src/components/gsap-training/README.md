# 🎯 GSAP Training - Exercícios Práticos

## 📚 Estrutura do Treinamento

### Nível 1: Fundamentos (Semana 1-2)

- **Exercício 1**: Animações básicas (to, from, fromTo)
- **Exercício 2**: Propriedades essenciais (x, y, scale, rotation, opacity)
- **Exercício 3**: Easing e duração
- **Exercício 4**: Delays e callbacks

### Nível 2: Timelines (Semana 3-4)

- **Exercício 5**: Timeline básica
- **Exercício 6**: Sequenciamento e posicionamento
- **Exercício 7**: Timeline com controles
- **Exercício 8**: Animações em loop

### Nível 3: Interatividade (Semana 5-6)

- **Exercício 9**: Hover effects
- **Exercício 10**: Click animations
- **Exercício 11**: Scroll-triggered animations
- **Exercício 12**: Mouse follow effects

### Nível 4: Avançado (Semana 7-8)

- **Exercício 13**: Morphing SVG
- **Exercício 14**: Text animations
- **Exercício 15**: Particle systems
- **Exercício 16**: Physics simulation

### Nível 5: Master (Semana 9-12)

- **Exercício 17**: Custom easing
- **Exercício 18**: WebGL integration
- **Exercício 19**: Performance optimization
- **Exercício 20**: Complete interactive scene

## 🚀 Como Usar

1. Comece pelo **TrainingGround.jsx** - é seu laboratório
2. Cada exercício tem:
   - 📝 Explicação do conceito
   - 🎯 Objetivo específico
   - 💡 Dicas de implementação
   - 🏆 Desafio extra
3. Pratique cada exercício até dominar
4. Modifique e experimente suas próprias variações

## 💪 Dicas de Estudo

- **Pratique diariamente** (mesmo que 30min)
- **Codifique junto** com os exemplos
- **Experimente variações** dos exercícios
- **Documente** o que aprender
- **Compartilhe** suas criações

Vamos começar! 🎉

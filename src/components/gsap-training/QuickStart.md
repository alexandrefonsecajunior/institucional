# 🚀 Quick Start - GSAP Training

## Como Acessar o Treinamento

### Opção 1: URL Direta

Acesse seu site com: `http://localhost:5173/#training`

### Opção 2: Query Parameter

Acesse: `http://localhost:5173/?training=true`

### Opção 3: Botão no Portfolio

Adicione um botão no seu portfólio que leva para o treinamento (implementar se desejar).

## 🎯 O que Você Vai Aprender

### Exercícios Incluídos:

1. **Animações Básicas** - `to()`, `from()`, `fromTo()`
2. **Propriedades Essenciais** - x, y, scale, rotation, opacity, skew
3. **Easing & Duração** - Curvas de animação e timing
4. **Delays & Callbacks** - Controle de timing e eventos
5. **Timeline Básica** - Sequências e controles

### Mais Exercícios Disponíveis:

- Hover Effects
- Scroll Animations
- Text Animations
- SVG Morphing
- Particle Systems
- Custom Easing
- Performance Optimization

## 📚 Como Usar

1. **Escolha um exercício** na interface
2. **Leia a explicação** do conceito
3. **Teste os exemplos** clicando nos botões
4. **Analise o código** fornecido
5. **Modifique e experimente** suas variações
6. **Complete o desafio** proposto

## 🔥 Dicas de Estudo

- **Pratique diariamente** (30min mínimo)
- **Codifique junto** - não apenas observe
- **Experimente variações** dos exemplos
- **Leia a documentação** oficial do GSAP
- **Compartilhe** suas criações

## 🌟 Próximos Passos

Após completar os 5 exercícios básicos, você pode:

1. Implementar os exercícios mais avançados
2. Criar suas próprias animações
3. Integrar GSAP ao seu portfolio
4. Explorar plugins como ScrollTrigger, MorphSVG, etc.
